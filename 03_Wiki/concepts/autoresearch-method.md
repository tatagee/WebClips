# 自动研究法 (Autoresearch Method)

## 定义
Karpathy 提出的利用 LLM 进行增量探索与知识积累的策略，常结合 Claude Code 使用。

## 核心要点
- 该概念由多篇最新文章综合得出。
- 在当前技术语境下，它解决了规模化扩展的核心瓶颈。

## 实践指南
给模型一个极其开放的研究目标，允许其通过自动化脚本抓取、提炼，并在多轮迭代中收敛结论。
